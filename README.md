# UEXChatBot
Prérequis:
* NodeJS >= 21

Installation:
* cd UEXChatBot
* npm install esm
* npm install tmi
* npm install node-fetch

Configuration:
* Récupérer une clé API sur https://uexcorp.space/api.html
* Connecter le bot à Twitch sur https://twitchapps.com/tmi/
* On met tout dans le settings.json (qu'il faut créér)
